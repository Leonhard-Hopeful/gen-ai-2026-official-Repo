# Exercises — 1.1 Python Programming 2 Intermediate

Complete these challenges to strengthen your understanding and build confidence.

## 1. Data Structures

Create a list of items, a dictionary of settings, and a set of unique values. Explain when each makes sense.

## 2. Functions and Modular Code

Write a function that computes the average of a list of numbers and include a docstring.

## 3. Modules and Packages

Split a simple script into two modules and import one function from the other.

## Bonus exercise
Create a short notebook or lesson outline that connects all three topics in this module. Include one real-world example and one teaching tip.
