# Exercises — 0.0 Git Vcs 2 Intermediate

Complete these challenges to strengthen your understanding and build confidence.

## 1. Branching and Merging

## 2. Git Workflows for Teams

## 3. Code Review and Quality

Draft a short code review checklist for reviewing another student’s AI code change.
