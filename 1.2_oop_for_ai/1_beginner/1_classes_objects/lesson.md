# Classes and Objects

## Overview

Meet the building block of object-oriented programming: classes define structure and objects bring them to life.

## Why this matters

This lesson explains a key concept and connects it to real AI course design and student success.

## Key concepts

- Class syntax
- Creating instances
- The role of self
- Constructors

## Example

```python
class Student:
    def __init__(self, name):
        self.name = name
```
