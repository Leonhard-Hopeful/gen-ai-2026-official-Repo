# Gen AI Full Course

A structured learning repository for an AI course covering:

- `0.0_git_vcs`: Git version control fundamentals, collaboration, workflows, and release practices.
- `0.1_env_setup`: Python environment setup, virtual environments, dependency management, containerization, and reproducible pipelines.
- `1.1_python_programming`: Python programming concepts from variables through advanced scripting.
- `1.2_oop_for_ai`: Object-oriented programming concepts tailored for AI design, including classes, inheritance, composition, and model architecture.
- `1.3_mathematics_for_ai`: Core mathematics for AI including vectors, linear equations, statistics, matrices, calculus, probability, and optimization.

Each section includes:

- `lesson.md` content per topic
- `exercises/exercises.md` for hands-on practice
- `kawood.md` quizzes for knowledge checks
